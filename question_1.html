Bug Localization Question

The problem is that JabRef is not treating the "--aux" argument properly.

From JabRef documentation

--aux infile[.aux],outfile[.bib] base-BibTeX-file

Extract a subdatabase from a .aux file

In other words, the --aux argument is used as input, and a file ending in extension "bib" is created.

A bug exists such that when the --aux argument is specified, the data that is imported is not written to the file system.

Please choose the code segment that when modified will fix the bug.
